Create a new type of product in WooCommerce as a combined product or service package.

== Description ==

[SW Product Bundles](http://wpthemego.com/) is a plugin that helps you to **create a new type of product in WooCommerce**.  It will include several products or services as a combined product or service package.

With this plugin, you can **bundle assemble products or services as a group** and offer a discount. As a result, you can promote customer to buy more to boost your sales.

== Features ==

* Allow to create product as bundles
* Shortcode to display bundled products as slider on homepage
* Support Hot Combo Slider
* Compatible with WordPress
* Compatible with WooCommerce
* Compatible with Visual PageBuilder
* Allow to choose item to display products
* Control the number of product displaying
* Allow to set interval time out for slider
* Allow to set speed for slider
* Fully responsive
* Translation ready
* Multi-language Support

== Frequently Asked Questions ==

= How to Install SW Product Bundles? =

<h4>1. Automatic installation:</h4>

This is the simplest way to install SW Product Bundles. Just after a few clicks, the plugin will be ready to use in your site.

<ol>
	<li>Login to your website and go to the Plugins.</li>
	<li>Click the Add New button.</li>
	<li>Type 'SW Product Bundles' in the search field of WordPress Plugin Directory.</li>
	<li>Select 'SW Product Bundles' then click Install Now button.</li>
	<li>After the plugin was installed successfully, click on Activate button to activate it.</li>
</ol>

<h4>2. Manual installation 1:</h4>
This is manual installation via Upload file path in the WordPress Admin Panel. Please follow steps as below:
<ol>
<li>Login to your website and go to the Plugins.</li>
<li>Click the Add New button.</li>
<li>Under Install Plugins, click the Upload Plugin.</li>
<li>Select the plugin zip file (sw-product-bundles_vx.x.x) from your computer then click the Install Now button.</li>
<li>You should see a message stating that the plugin was installed successfully.</li>
<li>Click the Activate Plugin.</li>
</ol>

<h4>2. Manual installation 2:</h4>
This is the manual installation. The plugin file will be uploaded into the server of the site domain. After that, the plugin will be activated in the Admin Panel. Please follow steps as below:
<ol>
<li>You should have access to the server where WordPress is installed.</li>
<li>Upload the plugin zip file (sw-product-bundles_vx.x.x) up to your server path(/wp-content/plugins/) and unzip it here.</li>
<li>Login to your website and go to the Plugins section of your admin panel.</li>
<li>Look for "SW Product Bundles" and click Activate.</li>
</ol>
