== Description ==

Shopee Integration for WooCommerce connects the woocommerce store with the Shopee marketplace by synchronizing the inventory, price, and other product details for the product creation and its management between the WooCommerce store and the Shopee marketplace.

A few notes about the sections above:

*   Profile Based Product Upload
*   Auto Sync Price and Inventory
*   Auto Import Products

= Minimum Requirements =

* WordPress 4.7 or greater
* WooCommerce 3.0.0 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater
